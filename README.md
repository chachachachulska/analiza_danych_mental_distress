# Mental Health Gen Z Analysis

This project analyzes Gen Z mental-health patterns by combining CDC SMART 2022 health data, Prolific survey responses, historical Facebook college rollout data, IPEDS institution metadata, and US Census region mappings.

## Research Thesis

The working thesis is that Gen Z mental-health outcomes are shaped by macro-environmental stressors rather than by a simple standalone "smartphone effect." At the US state level, the analysis tests whether mental distress varies with socioeconomic conditions, physical health context, device usage, and historical social-media rollout timing.

Proposed framing:

> The spatial distribution of youth distress: assessing structural predictors of Gen Z mental health across regional socioeconomic landscapes.

The Facebook rollout variable is used as a historical proxy for early social-media penetration in each state's college ecosystem. It should not be interpreted as direct evidence that today's Gen Z respondents personally adopted Facebook earlier.

## Current Findings

- Regional differences appear in CDC frequent mental distress rates, with the South showing higher average distress and the Northeast lower average distress in the current analysis.
- Prolific PHQ-9 depression rates are compared with CDC frequent mental distress rates to check whether both sources show similar state-level patterns.
- Mobile/device usage shows some association with distress in simple plots, but regression results suggest it is intertwined with economic variables.
- In the current robustness notebook, state-level income inequality remains statistically significant in the full model, while mobile usage does not.
- Physical health measures, including bad physical-health days and sleep duration, are included as contextual modifiers rather than treated as purely digital effects.

These findings are exploratory and state-level. They support hypothesis generation, not individual-level causal claims.

## Data Sources

- CDC SMART 2022 health data: mental-health days, physical-health days, sleep, BMI, income, and frequent mental distress for ages 18-25.
- Prolific mental-health survey: PHQ-9, GAD-7, demographics, respondent state, and survey device metadata for Gen Z respondents.
- Facebook introduction dates: college-level Facebook join dates and expansion batches from 2004-2005.
- IPEDS institution metadata: maps colleges to states.
- US Census regions: maps states to Census regions and divisions.

Local data files live under `data/`, but that directory is ignored by Git. The local DuckDB database is also generated locally and ignored.

## Prerequisites

- [Python 3.12+](https://www.python.org/downloads/)
- [uv](https://github.com/astral-sh/uv) for dependency management

## Setup and Replication

1. Install dependencies:

   ```bash
   uv sync
   ```

2. Prepare local data files under `data/`.

   The current ingestion script expects Parquet files such as:

   ```text
   data/cdc_smart_data_2022.parquet
   data/mental_health_survey.parquet
   data/intro_dates.parquet
   data/ipeds_hd.parquet
   data/regions.parquet
   ```

3. Ingest data into DuckDB:

   ```bash
   uv run python src/ingest.py
   ```

4. Run the analysis:

   ```bash
   uv run python src/analyze_genz.py
   ```

   This creates the derived summary tables used by the plots, including `prolific_genz_individual`, `prolific_state_summary`, and `cdc_state_summary`.

5. Generate the core visualizations:

   ```bash
   uv run python src/generate_visualizations.py
   ```

   Generated charts are written to `output/`, which is ignored by Git.

## Interactive Analysis

The project includes `notebook.ipynb` for exploratory robustness checks. It focuses on the inequality hypothesis and compares CDC mental distress against Prolific depression rates.

The notebook-generated plots are separate from the six core plots generated by `src/generate_visualizations.py`.

## Project Structure

- `src/ingest.py`: loads local Parquet files into DuckDB.
- `src/analyze_genz.py`: filters Gen Z respondents and creates summary tables.
- `src/generate_visualizations.py`: creates the six core output figures.
- `notebook.ipynb`: exploratory regression and robustness notebook.
- `queries/`: SQL scratch queries.
- `data/`: local input data and generated DuckDB database, ignored by Git.
- `output/`: generated plots, ignored by Git.
